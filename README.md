# Computing Hub — University of Roehampton

A UX-centred departmental web portal prototype for Computing students at the
University of Roehampton. Built as the artefact for my MSc Web Development
final project (2025–26).

## The problem
Departmental information is fragmented across Moodle, NEST, email and unlinked
webpages. In my survey (n = 31), 84% of students had missed an event, deadline
or opportunity because of it.

## The project
Following a mixed-methods, user-centred design process — comparative analysis
of four university platforms, interviews, survey, wireframing and task-based
usability testing with 13 participants — I designed and built this front-end
MVP.

## Tech
- HTML, CSS and JavaScript (no frameworks, no backend)
- 14 pages sharing one stylesheet
- WCAG 2.1 AA aligned
- Prototyped in Figma
